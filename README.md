Виселица (угадай слово)
Описание: Игрок пытается угадать слово, предложенное
компьютером, по буквам. За каждую ошибку игроку начисляетсяштраф, и он приближается к проигрышу. Особенности реализации: Список или массив для хранения
состояния угаданного слова, функции для проверки букв иобновления состояния, обработка ввода пользователя. Основные элементы: Управление строками, условные операторыдляпроверки попаданий и промахов.
